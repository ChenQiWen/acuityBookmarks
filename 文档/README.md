## 🎯 **文档使用指南**

### **按需求查找文档**

1. **了解项目** → 查看`开发指南/项目说明.md`
2. **开始开发** → 查看`开发指南/前端开发指南.md`或`后端开发指南.md`
3. **使用UI组件** → 查看`开发指南/UI组件库说明.md`
4. **了解迁移历程** → 查看`UI系统迁移/`目录
5. **解决技术问题** → 查看`技术修复报告/`目录
6. **项目规划参考** → 查看`项目管理/`目录
7. **产品介绍** → 查看`产品文档/`目录

### **文档更新原则**

- ✅ **新文档**: 按分类放入对应目录，使用中文命名
- ✅ **技术修复**: 记录到`技术修复报告/`
- ✅ **项目变更**: 记录到`项目管理/`
- ✅ **历史文档**: 移入`历史存档/`保留参考
