# AcuityBookmarks 快捷键说明

## 📋 所有快捷键

| 功能             | Mac       | Windows/Linux | 说明                               |
| ---------------- | --------- | ------------- | ---------------------------------- |
| 激活扩展         | Alt+A     | Alt+A         | 打开/关闭 popup                    |
| 书签管理         | Alt+B     | Alt+B         | 打开书签管理页面                   |
| 打开设置         | Alt+S     | Alt+S         | 打开设置页面                       |
| **快速添加书签** | **Alt+K** | **Alt+K**     | **快速添加书签（AI 智能分类）** ✨ |

## ⚠️ 避免快捷键冲突

### Chrome 原生快捷键（不要使用）

| 快捷键 | Chrome 原生功能    | 冲突          |
| ------ | ------------------ | ------------- |
| ⌘D     | 添加书签           | ❌ 已避免     |
| ⌘⇧D    | 添加所有标签为书签 | ❌ 已避免     |
| ⌘B     | 显示/隐藏书签栏    | ⚠️ 可能冲突   |
| ⌘⇧B    | 显示/隐藏书签栏    | ❌ **冲突！** |

### 我们的选择：Alt+K

**为什么选择这个？**

1. ✅ **格式一致**：与我们其他快捷键（Alt+A/B/S）保持一致，符合用户习惯
2. ✅ **简单可靠**：避免复杂的组合键导致的 manifest 解析错误
3. ✅ **不冲突**：Chrome 原生不使用 Alt+K
4. ✅ **易记易按**：K = "快"速添加（Quick Add），键盘位置方便
5. ✅ **跨平台统一**：Mac 和 Windows 使用相同的快捷键

**备选方案（如果需要）：**

- `F9` / `F10` - 功能键（完全安全）
- 用户可在 `chrome://extensions/shortcuts` 手动修改

## 🎯 使用建议

### 推荐使用方式

**方式 1：快捷键（最快）**

```
在任意网页 → 按 Command+Shift+B
→ 弹出对话框 → AI 自动建议 → 确认
```

**方式 2：右键菜单（最直观）**

```
页面右键 → "添加到书签..."
→ 弹出对话框 → AI 自动建议 → 确认
```

**方式 3：右键链接（精准）**

```
链接右键 → "添加到书签..."
→ 弹出对话框（自动填充链接地址）
```

## 🔧 如何修改快捷键

如果你想使用其他快捷键：

1. 打开 `chrome://extensions/shortcuts`
2. 找到 "AcuityBookmarks"
3. 点击编辑按钮
4. 输入你想要的快捷键组合

**建议的替代方案：**

- `Command+Option+D`
- `Command+Ctrl+D`
- `F9` / `F10`

## ⚡ 快速入门

**首次使用：**

1. 安装扩展后，打开 `chrome://extensions/shortcuts`
2. 确认快捷键已注册
3. 在任意网页按 `Command+Shift+B` 测试
4. 应该看到我们的自定义对话框（带 AI 建议）

**如果没反应：**

1. 检查扩展是否启用
2. 重新加载扩展（chrome://extensions/ → 刷新按钮）
3. 检查是否有错误
