# Avatar 头像组件

一个头像组件，用于显示用户头像或占位符。

## ✨ 特性

- 🖼️ **图片支持** - 显示用户头像
- 🔤 **文字头像** - 显示用户名首字母
- 📏 **多种尺寸** - sm、md、lg 等
- 🎨 **颜色定制** - 支持自定义背景色
- 📦 **组合组件** - 可能包含 Icon

## 📦 安装

```typescript
import { Avatar } from '@/components'
```

## 🔗 依赖组件

本组件依赖以下基础组件：

- [Icon](../../base/Icon/README.md)


## 🎯 基础用法

### 默认用法

```vue
<script setup lang="ts">
import { Avatar } from '@/components'
</script>

<template>
  <Avatar />
</template>
```

## 💡 使用场景

### 用户头像

```vue
<template>
  <Avatar src="/avatar.jpg" alt="用户名" />
</template>
```
