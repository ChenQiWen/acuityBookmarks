# Tooltip 提示框组件

一个提示框组件，用于显示额外的信息。

## ✨ 特性

- 📍 **多方向** - 支持上下左右四个方向
- ⚡ **自动定位** - 智能调整位置避免溢出
- 🎨 **样式定制** - 支持自定义样式
- ⏱️ **延迟显示** - 支持延迟显示和隐藏
- 📦 **零依赖** - 纯原子组件

## 📦 安装

```typescript
import { Tooltip } from '@/components'
```

## 🎯 基础用法

### 默认用法

```vue
<script setup lang="ts">
import { Tooltip } from '@/components'
</script>

<template>
  <Tooltip />
</template>
```
