# Divider 分割线组件

一个简单的分割线组件，用于分隔内容区域。

## ✨ 特性

- 📏 **方向支持** - 水平和垂直分割线
- 🎨 **样式定制** - 支持实线、虚线等
- 📝 **文字分割** - 支持带文字的分割线
- 📦 **零依赖** - 纯原子组件

## 📦 安装

```typescript
import { Divider } from '@/components'
```

## 🎯 基础用法

### 默认用法

```vue
<script setup lang="ts">
import { Divider } from '@/components'
</script>

<template>
  <Divider />
</template>
```
