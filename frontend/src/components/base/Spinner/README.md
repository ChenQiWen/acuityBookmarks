# Spinner 加载动画组件

一个简洁的加载动画组件，用于显示加载状态。

## ✨ 特性

- 🎨 **多种样式** - 支持不同的加载动画样式
- 📏 **多种尺寸** - sm、md、lg 满足不同场景
- 🌈 **颜色定制** - 支持主题色
- ⚡ **性能优化** - 使用 CSS 动画，性能优异
- 📦 **零依赖** - 纯原子组件

## 📦 安装

```typescript
import { Spinner } from '@/components'
```

## 🎯 基础用法

### 默认用法

```vue
<script setup lang="ts">
import { Spinner } from '@/components'
</script>

<template>
  <Spinner />
</template>
```
