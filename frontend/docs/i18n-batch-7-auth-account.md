## 翻译说明

### 类别说明
本批次翻译的是认证与账户（Authentication & Account）相关的文案，主要包括：
- 登录/注册表单（email, password, login, register）
- 密码管理（forgot_password, reset_password, password_requirements）
- 邮箱验证（email_verification, verification_success）
- 社交登录（login_with_google, login_with_microsoft）
- 账户状态（login_success, session_expired, account_locked）
- 错误提示（invalid_email, password_mismatch）

### 翻译原则
1. **用户友好**：认证流程的文案要清晰、友好
2. **安全提示**：密码相关的提示要明确、专业
3. **一致性**：与行业标准保持一致（如"登录"vs"登入"）
4. **简洁明了**：避免冗长的描述

### 特殊说明
- `auth.` 前缀表示这些是认证相关的文案
- 密码强度（weak/medium/strong）使用简短的形容词
- 社交登录按钮保持品牌名称不翻译（Google, Microsoft, GitHub）
- 错误提示要明确指出问题所在

## 使用示例

```vue
<template>
  <form @submit="handleLogin">
    <Input 
      v-model="email" 
      :label="t('auth.email')" 
      :placeholder="t('auth.enter_email')" 
    />
    <Input 
      v-model="password" 
      type="password"
      :label="t('auth.password')" 
      :placeholder="t('auth.enter_password')" 
    />
    <Button type="submit">{{ t('auth.login') }}</Button>
    <a @click="showForgotPassword">{{ t('auth.forgot_password') }}</a>
  </form>
</template>

<script setup lang="ts">
import { t } from '@/utils/i18n-helpers'

const handleLogin = async () => {
  try {
    await login(email.value, password.value)
    showToast(t('auth.login_success'))
  } catch (error) {
    showError(t('auth.login_failed'))
  }
}
</script>
```
