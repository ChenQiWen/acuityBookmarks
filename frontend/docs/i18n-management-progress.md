# Management.vue i18n 修复进度

## 概述

Management.vue 是一个大型组件（3172行），包含大量硬编码中文文本。本文档记录 i18n 修复的进度。

## 已完成 ✅

### 1. 翻译键添加
- ✅ 在 `en/messages.json` 中添加了约100个 Management 相关的翻译键
- ✅ 在 `zh_CN/messages.json` 中添加了对应的中文翻译

### 2. 代码修改
- ✅ 导入 `t` 函数：`import { t } from '@/utils/i18n-helpers'`
- ✅ 替换常量：
  - `MSG_CANCEL_EDIT`
  - `MSG_CANCEL_ADD`
- ✅ 替换 computed 属性：
  - `addDialogTitle`
  - `addConfirmText`
  - `applyButtonTooltip`
  - `clearSelectionTooltip`
  - `deleteButtonTooltip`
- ✅ 替换初始化值：
  - `traitDetectionProgress.message`
  - `organizeProgress.message`
- ✅ 替换 template 文本：
  - 特征检测对话框标题
  - AI 整理对话框标题
  - 统计信息标签（新增文件夹、新增书签、移动、编辑、删除、总计）
  - AI 生成标记
  - 左侧面板标题"我的书签"
  - 展开/收起按钮 tooltip
  - 应用按钮文本
  - AI 整理按钮文本和状态

## 待完成 ⏳

由于文件过大（3172行），以下区域仍包含硬编码中文，需要后续批量处理：

### Template 部分
1. **对话框内容**：
   - 详细操作列表标题
   - 分组汇总内容
   - 大规模更改警告文本
   - 应用进度对话框内容
   - 编辑书签对话框
   - 批量删除确认对话框
   - 编辑文件夹对话框
   - 删除文件夹确认对话框
   - 添加新项目对话框

2. **批量操作栏**：
   - "已选择" / "已全选"
   - "条书签" / "个文件夹"
   - "清除选择"
   - "删除"按钮

3. **表单标签**：
   - "书签标题" / "书签链接"
   - "文件夹标题" / "文件夹名称"
   - "标题" / "链接地址"

### Script 部分
1. **通知消息**（notificationService.notify）：
   - 成功消息：书签已添加、已更新、已删除、已移动等
   - 错误消息：添加失败、编辑失败、删除失败等
   - 警告消息：无效URL、内部链接等
   - 信息消息：无更改、外部变更检测等

2. **日志消息**（logger.info/error/warn）：
   - 这些主要用于调试，可以保持中文或改为英文

3. **函数内字符串**：
   - 错误提示文本
   - 确认对话框消息
   - 进度提示文本

## 建议的完成方式

### 方案 1：手动逐步替换（推荐）
按优先级逐步替换：
1. 高优先级：用户直接可见的 UI 文本（对话框、按钮、标签）
2. 中优先级：通知消息和提示文本
3. 低优先级：日志消息（可选，保持中文也可以）

### 方案 2：自动化脚本
创建一个脚本来：
1. 扫描所有硬编码中文字符串
2. 自动生成翻译键
3. 自动替换为 `t()` 调用

## 测试计划

完成替换后需要测试：
1. ✅ 切换到英文语言，检查所有文本是否正确显示

3. ✅ 测试所有对话框和通知消息
4. ✅ 运行类型检查：`bun run typecheck`
5. ✅ 运行代码检查：`bun run lint`
